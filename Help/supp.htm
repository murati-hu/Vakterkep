A Vaktérkép támogatása

A Vaktérkép teljes mértékben kompatibilis a 32 bites illetve 64 bites Windows rendszerekkel. A program társítható a vaktérkép dokumentumokkal(*.vtk), így a dokumentum futtatásakor automatikusan betöltõdik a Vaktérkép

Rendszerkövetelmények

Operációs rendszer: Microsoft Windows 95/98/ME/NT/2000
RAM: min.16 MB; ajánlott 32MB
CPU: min. 166 MHz; ajánlott: 266MHz
Rendszerfájlok:
A program Microsoft Visual Basicben íródott, így a futásához szükségesek a VB Runtime library-k.
 STDOLE2.TLB
OLEPRO32.DLL
OLEOUT32.DLL
MSVBVM60.DLL
COMCAT.DLL
ASYCFILT.DLL
COMDLG32.OCX - A Windows beépített párbeszédablakának kezeléséhez.
